PROJECT 0
This project is written for linux/unix systems, and is not expected to work in any other environment.
How to install:
1. Copy the folder to your desired destination
2. Run the make command to process the makefile.

USAGE
program name: p0
From bash terminal, execute the program using the following command:
./p0

The above will give you a prompt to write content manually.

OTHER ARGUMENTS
Stream a file into the application:
./p0 < somefile.txt

NOTE: The above argument produces output files called 'output'.

Load a file into the application:
./p0 somefile.txt

NOTE:  The above argument produces output files that have the begin with the name of the file from your command line argument.
